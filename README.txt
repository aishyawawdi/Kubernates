In this Folder you will find TWO *Deployments*:
1- deployment for Apache.
2- deployment for mysql.

you will also find adequate *Services* for each deployment.

Requirements:
1- you will need to install kubernates ( you can find out how on this link: https://kubernetes.io/docs/tasks/tools/install-kubectl/).
2- install kubectl for your machine.

how to RUN:
Run out the command "kubectl apply -f ." in order to run all the YML files in your directory.
